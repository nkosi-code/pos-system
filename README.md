# POS System

A Basic Point of Sale (POS) system built using ASP.NET Core and React.

## Features
- Add products
- Delete products
- Sell products
- Track total sales
- Calculate total revenue
- Dashboard overview

## Tech Stack
- Frontend: React
- Backend: ASP.NET Core Web API
- Database: SQL Server
